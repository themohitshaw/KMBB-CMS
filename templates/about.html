Project Overview

College Management and Attendance System developed using Flask and MySQL. It helps manage attendance, student records, and notices digitally.

Key Features

  • Online Attendance
  • Notice Management
  • Student Data Storage
  • Download Notes
  • Secure Login System

Technologies Used

  • Frontend: HTML, CSS, Bootstrap
  • Backend: Flask
  • Database: MySQL
